# Support

Status: Active

## Before Opening An Issue

1. Read `README.md`.
2. Run `scripts/validate-regulation-index.*` and `scripts/tests/run-regulation-tests.*` when changing this shelf.
3. Confirm whether the problem is usage, defect, or documentation.

## Where To Ask

- reproducible defects: open a bug report
- feature proposals: open a feature request
- security issues: follow `SECURITY.md`

## What To Include

- command used
- platform (Windows PowerShell or Linux/macOS bash)
- expected behavior
- actual behavior
- smallest useful log or output excerpt
